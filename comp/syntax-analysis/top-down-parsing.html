Recursive-Descent Parsing Algorithm
Choose an A-production, A -> X1 X2 ... Xk;
for i = 1 to k:
  if Xi is a nonterminal:
    call procedure Xi();
  else if Xi equals the current input symbol a:
    advance the input to the next symbol;
  else:
    raise an error
